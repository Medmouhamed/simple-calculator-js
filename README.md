# Simple JavaScript Calculator

A clean and functional web-based calculator built using vanilla JavaScript, HTML5, and CSS3.

## Features
* **Basic Arithmetic**: Supports addition, subtraction, multiplication, and division.
* **Responsive Design**: Works on desktop and mobile screens.
* **User-Friendly Interface**: Minimalist and intuitive layout.

## Technologies Used
* [cite_start]**HTML5**: For semantic structure. 
* [cite_start]**CSS3**: For styling and layout. 
* [cite_start]**JavaScript**: For calculation logic and DOM manipulation. 

## How to Run
1. Clone or download the repository.
2. [cite_start]Open `calculator.html` in any modern web browser.
